pertama kita ketik perintah git init lalu kita buat sebuah file dalahm folder yang kita buat

setelah file yg kita buat jadi kita bisa mengecek status file tersebut menggunakan perintah git status

setekah kita cek kita tambahkan file tersebut dengan menggunakan periknath git add . dan buat new reposity baru lalu salin link repoaitiy baru itu dan ketikan perintah pada git

ketikan perintah pada git dengan perintah git remote add origin url (isi dengan url yang kita salin tadi )

setelh itu kita publidh ke git hub dengan cara ketik perintah git push -u origin master
